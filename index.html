“Something 101” 是什么意思？

它通常是指“某事情的入门介绍”。这种说法的典故来源于美国教学系统中，大学初级课程代码101。课程代码101通常表示这是入门课程，没有其他的前提要求。

JS101 是 JavaScript 的新手课堂，收集并翻译了 jQuery 学习中心与 DailyJS 的 JavaScript 101教程。如果你是 JavaScript 新手，那就从这里开始吧。
